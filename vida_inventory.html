Request:

We would like to end up with as simple a structure as possible, eliminating from our inventory all irrelevant files, replacing “sample” files with true raw data, and standardizing the names of each dataset, location, period and format.

To this end, we will map all the current data by populating the below table. The meanings of each of the fields in the table are as follows:

  • Country: Self-explanatory
  • Year(s): Self-explanatory
  • Name: The colloquial name to be used when referring to this dataset
  • Age group: The age group of the observations in the dataset
  • File name: The exact file name of the dataset (only one file per row)
  • File location: A unique url from which the file can be downloaded (by authorized collaborators only), such as a google drive or dropbox link
  • File type: Whether the file in question is “Raw data” (ie, the data as stored by the site) or “Dictionary” (ie, meta-data, or supporting information to decode the raw data)
  • Comment: A open-text field for any relevant comments
